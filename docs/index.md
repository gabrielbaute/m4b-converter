# 📖 M4B Converter

**Herramienta CLI para convertir y optimizar audios a M4B**

## Características

- ✅ Conversión a M4B desde MP3, WAV, FLAC, etc.
- 🔄 Optimización de archivos M4B existentes
- 📝 Metadatos personalizados
- 🖼️ Extracción de portadas
- 📊 Barra de progreso interactiva
- 📦 Procesamiento por lotes

## Instalación Rápida

```bash
pip install m4b-converter
```

## Uso Básico

```bash
# Convertir un archivo
m4b convert audio.mp3 --bitrate 64k

# Analizar un archivo
m4b analyze audio.mp3

# Procesar directorio
m4b batch ./audiolibros/
```
