# Instalación

## Requisitos

- Python 3.14 o superior
- FFmpeg instalado en el sistema

## Instalar FFmpeg

### Windows
```bash
# Con Chocolatey
choco install ffmpeg
```

### macOS
```bash
brew install ffmpeg
```

### Linux
```bash
sudo apt install ffmpeg  # Ubuntu/Debian
```

## Instalar M4B Converter

```bash
pip install m4b-converter
```

## Verificar

```bash
m4b version
```
